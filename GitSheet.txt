# Git Cheat Sheet

## Navigate and Check Status
- `cd path/to/your/project`  
- `git status`

## Stage Files for Commit
- `git add filename`  
- `git add .`  
- `git add -p`  (stage interactively)

## Commit Your Changes
- `git commit -m "Short, clear commit message"`

## Push Commits to Remote
- `git push origin main`

## Pull Latest Changes from Remote
- `git pull origin main`

## Branching
- `git checkout -b feature-name`  (create and switch to branch)  
- `git checkout main`  (switch to main)  
- `git branch`  (list branches)  
- `git branch -d feature-name`  (delete branch)

## Merge Branch into Main
- `git checkout main`  
- `git merge feature-name`

## Collaborating Tips
- Add collaborators via GitHub repo settings  
- Pull regularly: `git pull origin main`  
- Push changes with good commit messages

## Undo and Fix Mistakes
- `git restore filename`  (discard unstaged changes)  
- `git restore --staged filename`  (unstage a file)  
- `git reset HEAD~1`  (undo last commit, keep changes staged)  
- `git reset --hard HEAD~1`  (undo last commit, discard changes â€” dangerous!)

## Helpful Info Commands
- `git log`  (show commit history)  
- `git diff`  (show unstaged changes)  
- `git diff --staged`  (show staged changes)  
- `git remote -v`  (show remote URLs)
